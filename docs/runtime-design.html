Design Notes and Current Limitations

Good

  • GPU-resident state with buffer-handle swaps.
  • Grid build and SPH force kernels authored in Python DSL.
  • Renderer can consume simulation buffers directly.

Caveats

  • Hash-grid build currently uses O(num_cells * N) scans.
  • Prefix sum kernel is single-thread serial.
  • Large particle counts may need parallel scan/scatter redesign.
